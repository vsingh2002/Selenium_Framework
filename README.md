__Selenium Registration Automation__

This project contains a simple Selenium automation script using Java, TestNG, and Firefox.
The script opens the Demo Web Shop website, fills the registration form with a randomly generated email ID, submits the form and validates the success message.

__Technologies Used__

**•** Java  
**•** Selenium WebDriver  
**•** TestNG  
**•** Firefox (GeckoDriver)  
**•** Maven



__How to Run__

**1.** Clone the project  
**2.** Import it as a Maven project  
**3.** Add GeckoDriver to your system PATH  
**4.** Run the `BasicScript` class


What the Script Does

• Opens browser  
• Navigates to registration page  
• Enters details with random email  
• Submits form  
• Verifies "Your registration completed" message
